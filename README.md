# MITRE ATT&CK Labs

A collection of hands-on MITRE ATT&CK Navigator exercises focused on threat actor analysis, defensive mitigations, and security framework mappings.

## Objectives

- Learn and apply the MITRE ATT&CK Framework
- Analyze adversary techniques and procedures (TTPs)
- Map defensive mitigations to ATT&CK techniques
- Visualize security data using ATT&CK Navigator
- Develop practical threat hunting and SOC analysis skills

## Skills Demonstrated

- MITRE ATT&CK Framework
- ATT&CK Navigator
- Threat Intelligence Analysis
- Adversary TTP Mapping
- Security Control Mapping
- Defensive Security Analysis
- Threat Hunting Concepts
- SOC Analyst Methodologies

## Tools Used

- MITRE ATT&CK Navigator
- MITRE ATT&CK Framework
- ATT&CK Data Sources

## Labs

- **Sable Bluff** – Analysis of APT29, DarkComet, and Active Directory Configuration mitigations.
